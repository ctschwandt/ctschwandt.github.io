Python-2 Interpreter

Stack: OCaml, ocamllex

GitHub
  • Implemented a dynamically typed Python-2 interpreter in OCaml supporting mixed-type arithmetic, variables, and assignment semantics.
  • Built a custom lexer and recursive-descent parser to generate expression and statement ASTs for Python control-flow and blocks.
  • Built a runtime environment with symbol tables to execute Python expressions, assignments, and control flow.
